Quick Start

Get started with photo verification in 3 simple steps:

1

Get your API Key

Use the test key test_key_12345 for development

2

Submit a Photo

POST to /api/v1/photos with your image

3

Check the Result

Get instant verification with confidence score

# Quick test with cURL
curl -X POST https://your-domain.com/api/v1/photos \
  -H "X-API-Key: test_key_12345" \
  -F "image=@photo.jpg"

Authentication

All API requests require authentication via API key. Include your key in the X-API-Key header:

X-API-Key: your_api_key_here

You can also pass it as a query parameter (not recommended for production):

GET /api/v1/stats?api_key=your_api_key_here

Test API Key

For development and testing, use:

test_key_12345

This key has a rate limit of 100 requests/hour

Rate Limits

API requests are rate limited based on your plan tier:

Tier Requests/Hour Bulk Limit
Developer 100 10 photos
Standard 1,000 50 photos
Enterprise 10,000 100 photos

Rate limit headers are included in every response:

X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 998
X-RateLimit-Reset: 1703836800

Submit Photo for Verification

POST /api/v1/photos

Submit a photo for cryptographic verification. Returns verification result with confidence score.

Request Body (multipart/form-data)

Field Type Required Description
image File Yes Image file (JPEG, PNG). Max 50MB
metadata JSON No Additional metadata (GPS, device info)
claim JSON No C2PA claim data from TrueShot app

Response

200 OK 
{
  "success": true,
  "data": {
    "photoId": "PH-lxyz123-a1b2c3d4",
    "verified": true,
    "confidence": 0.95,
    "recommendation": "APPROVE",
    "checks": {
      "signature": true,
      "merkleRoot": true,
      "metadata": true,
      "temporal": true
    },
    "fraudRisk": "LOW",
    "processingTimeMs": 245
  }
}

List Photos


                    

                        

                            GET
                            /api/v1/photos
                        

                        
List all verified photos for your account with pagination.


                        
Query Parameters

                        

                            
                                

                                    Parameter
                                    Default
                                    Description
                                

                            
                            
                                

                                    limit
                                    20
                                    Number of results (max 100)
                                

                                

                                    offset
                                    0
                                    Pagination offset
                                

                                

                                    verified
                                    -
                                    Filter by verified status (true/false)
                                

                                

                                    since
                                    -
                                    Filter by date (ISO 8601)

Verify by Hash


                    

                        

                            GET
                            /api/v1/verify/:hash
                        

                        
Check if an image hash exists in the verification database.

                    

                


                

                    
Bulk Verify


                    

                        

                            POST
                            /api/v1/bulk
                        

                        
Verify multiple photos in a single request (up to 50 photos).


                        
# Bulk upload example
curl -X POST https://your-domain.com/api/v1/bulk \
  -H "X-API-Key: your_api_key" \
  -F "images=@photo1.jpg" \
  -F "images=@photo2.jpg" \
  -F "images=@photo3.jpg"

Configure Webhook


                    

                        

                            PUT
                            /api/v1/webhook
                        

                        
Set or update your webhook URL to receive verification events.


                        
{
  "url": "https://your-server.com/webhooks/trueshot"
}

                    

                


                

                    
Webhook Events

                    
When configured, webhooks are sent for the following events:


                    

                        
photo.verified

                        
Sent when a photo verification is complete.

                        
{
  "event": "photo.verified",
  "data": {
    "id": "PH-lxyz123-a1b2c3d4",
    "verified": true,
    "confidence": 0.95,
    "createdAt": "2024-01-15T10:30:00Z"
  }
}

Recommendations

                    
The API returns one of the following recommendations based on confidence score:


                    

                        
                            

                                Recommendation
                                Confidence
                                Action
                            

                        
                        
                            

                                APPROVE
                                >= 90%
                                High confidence - safe to approve
                            

                            

                                APPROVE_WITH_REVIEW
                                70-89%
                                Likely authentic - quick review recommended
                            

                            

                                MANUAL_REVIEW
                                50-69%
                                Some concerns - manual review required
                            

                            

                                REJECT
                                < 50%
                                High fraud risk - reject or investigate
